# **Alert My Weather**


## **What this program does**
This program will access the `openweathermap` api to request weather data related to local (or non-local) towns.

It will then send the requested data to a specified email address, after being parsed for specific details chosen by the user.

### **Libraries**
- requests - https://docs.python.org/3/library/urllib.request.html
- smtplib - https://docs.python.org/3/library/smtplib.html
- os - https://docs.python.org/3/library/os.html
